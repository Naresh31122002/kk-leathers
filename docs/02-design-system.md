# 1. Purpose

This document defines the complete visual design language for KK Leathers.

Claude Code must strictly follow this document.

Never invent colors, spacing, typography, shadows or animations outside these specifications.

Consistency is more important than creativity.

Always highlight one primary product.

# 29. Background Rules

Backgrounds should remain dark.

Never use busy backgrounds.

---

# 30. Lighting Style

Soft studio lighting.

Luxury photography.

Gentle highlights.

Visible leather texture.

No harsh reflections.

---

# 31. Empty Space Philosophy

Whitespace is intentional.

Do not fill empty areas unnecessarily.

Luxury requires breathing room.

---

# 32. Animation Consistency

All animations should use a consistent easing curve.

Recommended:

power3.out

or

power2.inOut

Avoid inconsistent easing.

---

# 33. Component Consistency

Buttons

Cards

Images

Videos

Sections

Headings

All should follow identical spacing and alignment rules.

---

# 34. Responsive Behavior

Animations should adapt to screen size.

Never allow products to leave the viewport.

On mobile:

Reduce animation distance.

Reduce scale.

Keep interactions lightweight.

---

# 35. Accessibility

Maintain high contrast.

Support keyboard navigation.

Respect prefers-reduced-motion.

Provide alt text for every image.

Ensure focus indicators remain visible.

---

# 36. Performance Budget

Images

Use optimized WebP where transparency is not needed.

Transparent assets

PNG only.

Videos

MP4 H.264

Short duration

Compressed without noticeable quality loss.

---

# 37. Things Claude Must Never Do

Never redesign the brand colors.

Never change typography.

Never replace product images.

Never generate placeholder content.

Never add random icons.

Never use Bootstrap styling.

Never use Tailwind default blue accents.

Never use cartoon animations.

Never change the folder structure.

Never rename existing assets.

---

# 38. Future Scalability

The design system should support adding:
